# macOS Portfolio (Hannah MacDonald)

An interactive macOS-inspired portfolio built with Next.js and Tailwind CSS.

## Demo

- Live site: https://hannah-marie-macdonald.com

## Features

- Desktop + dock + draggable windows
- Light/dark wallpaper switching
- Spotlight / Launchpad UX
- Portfolio apps (Safari, Projects, Notes, Terminal, Analytics, etc.)

## Getting Started

### Prerequisites

- Node.js 18+ recommended

### Install & Run

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Where to Edit Content

- components/apps/safari.tsx: main “website” content
- components/apps/files.tsx: Projects window + hover descriptions
- components/apps/notes.tsx: Notes content
- components/apps/terminal.tsx: Terminal commands + outputs
- components/apps/vscode.tsx: Faux VS Code view
